# HardcorePlugin - 빌드 및 설치 가이드

## 📦 준비물
- JDK 21 이상 (https://adoptium.net)
- Maven 3.8 이상 (https://maven.apache.org/download.cgi)
- Paper 1.21.1 서버

---

## 🔨 빌드 방법

### 1. 이 폴더(HardcorePlugin)를 PC에 압축 해제

### 2. 터미널(cmd/PowerShell)에서 실행:
```bash
cd HardcorePlugin
mvn package
```

### 3. 빌드 완료 후 파일 위치:
```
HardcorePlugin/target/HardcorePlugin-1.0.0.jar
```

### 4. 서버 plugins 폴더에 복사 후 재시작

---

## ⚙️ 기능 목록

### ✅ 구현된 기능

| 기능 | 설명 |
|------|------|
| 🪦 사망 시 머리 드랍 | 플레이어 사망 시 자신의 머리가 드랍됨 |
| 🔊 사망 시 번개 소리 | 전체 서버에 trident thunder 사운드 재생 |
| 🚪 사망 시 로그아웃 | 하드코어 사망 시 자동 킥 처리 |
| ⏸ PAUSE 플레이어 | 체력 0 → 1 변환, 전체 채팅에 PAUSE 표시 |
| 💣 엔더크리스탈 무폭발 | 상호작용/공격 시 조용히 제거 |
| ⚓ 정박기 무폭발 | 오버월드/엔드에서 상호작용 시 조용히 제거 |
| 🏗️ 건축 높이 제한 | Y 601 초과 블록 설치 불가 |
| 🎫 부활권 제작 | 특수 재료로 제작, 비콘에 우클릭 사용 |
| 🔴 비콘 부활 의식 | 2분 유지 시 부활 완료, 파괴 시 드래곤 알 지급 |
| ⚒️ 커스텀 메이스 | 기본 메이스 제작법 삭제 후 새 제작법 등록 |
| 💛 커스텀 토템 | 레이드 드랍 없이 제작으로만 획득 |

## 📋 제작법 정리

### 🎫 부활권
```
[ 네더라이트주괴 ][ 네더의별    ][ 네더라이트주괴 ]
[ 플레이어머리   ][ 불사의토템  ][ 플레이어머리   ]
[ 네더의별       ][ 드래곤알    ][ 네더의별       ]
```

### ⚒️ 커스텀 메이스
```
[ Heavy Core ][ 막대      ][ Heavy Core ]
[ 네더블록   ][ Heavy Core][ 네더블록   ]
[ 막대       ][ Heavy Core][ 막대       ]
```
*(Heavy Core 1개, 네더블록 2개, 막대 5개)*

### 💛 커스텀 토템
```
[ 다이아블록  ][ 고대잔해  ][ 다이아블록  ]
[ 고대잔해    ][ 마황      ][ 고대잔해    ]
[ 다이아블록  ][ 고대잔해  ][ 다이아블록  ]
```
*(다이아블록 4개, 고대잔해 4개, 마황 1개)*

## 🛠️ 관리자 명령어

| 명령어 | 설명 |
|--------|------|
| `/hcpause add <플레이어>` | PAUSE 목록에 추가 |
| `/hcpause remove <플레이어>` | PAUSE 목록에서 제거 |
| `/hcpause list` | PAUSE 목록 확인 |
| `/hcrevive <플레이어>` | 부활권 직접 지급 |

## ⚠️ 참고 사항

- **레이드 불사의 토템 제거**: 이 기능은 플러그인만으로 완전히 막기 어려움.
  데이터팩으로 Pillager Raid 루트 테이블에서 `totem_of_undying` 제거 권장.
- **비콘 빔 색**: Minecraft API 특성상 정확한 "진홍색"은 유리 블록 색상으로 결정됨.
  비콘 위에 빨간 유리를 설치하면 진홍색 빔이 나옴 (자동 설치 기능 추가 가능).
- PAUSE 목록은 서버 재시작 후에도 `config.yml`에 저장되어 유지됨.
